# Architecture Diagrams

## Purpose

Conventions for creating and maintaining architecture diagrams for stage
READMEs and documentation.

## Guidelines

- Use Mermaid or SVG for architecture diagrams
- Store generated SVGs in `docs/assets/architecture/`
- Store demo screenshots and GIFs in `docs/assets/demos/`
- Keep diagrams aligned with implemented capabilities
- Do not hand-edit generated SVGs; update the source and regenerate
