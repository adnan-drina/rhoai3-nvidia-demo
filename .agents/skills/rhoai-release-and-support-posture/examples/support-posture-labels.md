# Support Posture Labels

Use these labels in README or implementation review notes.

```text
Support posture: GA
Source: RHOAI 3.4 release notes / supported configurations
Demo implication: acceptable as a normal demo dependency after implementation.
```

```text
Support posture: Technology Preview
Source: RHOAI 3.4 release notes
Demo implication: suitable for showcasing current direction, not production SLA.
```

```text
Support posture: Developer Preview
Source: RHOAI 3.4 release notes
Demo implication: unsupported experimental content; use only with explicit intent.
```

```text
Support posture: Removed
Source: RHOAI 3.4 release notes
Demo implication: do not use in active implementation.
```
