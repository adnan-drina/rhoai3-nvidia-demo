## Source Boundaries

- This source defines data scientist and AI engineer workflows in the catalog.
- It assumes catalog sources and model registry access are already configured.
- It does not define administrator catalog source governance.
- It does not define model registry database or permissions configuration.
- It does not fully define serving runtime configuration after the catalog
  deployment handoff.
